instala o vue.js:
npm install -g @vue/cli

ver versão do vue.js para verificar se está funcionando:
vue --version

criar projeto vue:
vue create <nome-do-projeto>

rodar aplicação:
npm run serve

rodar a aplicação em modo de produção:
npm run serve -- --mode production

build da aplicação:
npm run build

adiciona rotas automaticamente:
vue add routes

instala o pacode de rotas:
npm install vue-router
